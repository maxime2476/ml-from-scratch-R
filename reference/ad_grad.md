# Gradient d'une fonction scalaire par mode inverse

Gradient d'une fonction scalaire par mode inverse

## Usage

``` r
ad_grad(f, x)
```

## Arguments

- f:

  fonction `adnode -> adnode scalaire` (écrite avec les opérateurs
  ci-dessus).

- x:

  point (numérique) où évaluer le gradient.

## Value

le gradient \\\nabla f(x)\\ (même forme que `x`).
