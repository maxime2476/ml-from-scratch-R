# mlfromscratch : ML from Scratch in R

Réimplémentation pédagogique, en R base, des principaux modèles
d'apprentissage automatique, adossée à leurs dérivations mathématiques
et validée contre les packages de référence. Voir les dérivations dans
`derivations/` et le rapport de synthèse dans `rapport/`.
