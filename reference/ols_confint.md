# Intervalles de confiance des coefficients (éq. 1.8)

Intervalles de confiance des coefficients (éq. 1.8)

## Usage

``` r
ols_confint(object, level = 0.95)
```

## Arguments

- object:

  objet `ols`.

- level:

  niveau de confiance (défaut 0.95).

## Value

matrice à deux colonnes (bornes inf/sup).
