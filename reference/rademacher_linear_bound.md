# Borne théorique de Rademacher pour une classe linéaire (éq. 13.5)

\\\hat{\mathfrak R}\_S \le B\rho/\sqrt n\\, \\\rho=\max_i\\x_i\\\\.

## Usage

``` r
rademacher_linear_bound(X, B = 1)
```

## Arguments

- X:

  matrice n x d des points.

- B:

  rayon de la boule (défaut 1).

## Value

la borne \\B\rho/\sqrt n\\.
