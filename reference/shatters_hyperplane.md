# La classe des hyperplans pulvérise-t-elle un ensemble de points ? (Déf. 13.6)

Teste si les \\2^m\\ étiquetages d'un ensemble de m points sont TOUS
linéairement séparables (donc réalisables par un demi-espace).

## Usage

``` r
shatters_hyperplane(X)
```

## Arguments

- X:

  matrice m x d des points.

## Value

TRUE si l'ensemble est pulvérisé par les hyperplans de R^d.
