# Test de White (heteroscedasticite generale)

Breusch-Pagan augmente des carres et produits croises des regresseurs :
detecte toute forme d'heteroscedasticite liee a \\X\\.

## Usage

``` r
white_test(formula, data)
```

## Arguments

- formula, data:

  cf. `bp_test`.

## Value

liste : `statistic`, `df`, `p_value`.
